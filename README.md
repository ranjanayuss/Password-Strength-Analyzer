# Password Strength Analyzer

## Overview
This project evaluates the strength of user-entered passwords based on:

- Password length
- Uppercase letters
- Lowercase letters
- Numbers
- Special characters

It also suggests a stronger password if the entered password is weak.

## Features

- Password Strength Analysis
- Strength Meter
- Password Improvement Suggestions
- User-Friendly Interface

## Technologies Used

- HTML
- CSS
- JavaScript

## How to Run

1. Download the project files.
2. Open index.html in any modern browser.
3. Enter a password.
4. Click "Analyze Password".
